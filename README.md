# Portfolio


A premium, cinematic digital portfolio engineered for high-performance software showcasing. This project balances technical precision with visceral user experiences, leveraging the "Antigravity" design philosophy: light, depth, and modularity.

## 🌌 The Vision
This portfolio is not just a showcase; it's a technical statement. Built with a focus on sovereign technology and regional excellence, it serves as the digital architecture for **Sivchheng Kheang**, a Technical Lead at the **KOOMPI Collective**.

## ✨ Key Features

### 🎬 Cinematic Motion
- **Staggered Character Reveals**: Character-by-character entrance animations for hero titles using Framer Motion.
- **Micro-interactions**: Adaptive `whileTap` scaling and hover-reactive tokens for a tactile digital experience.
- **Smooth Orchestration**: Integrated **Lenis Smooth Scroll** for high-end, responsive parallax effects.

### 📱 Adaptive Architecture
- **Static Mobile Isolation**: A hardened mobile navigation system with 100% solid background isolation and full-screen scroll locking.
- **Fluid Layouts**: Responsive design optimized across iPhone SE, high-end tablets, and ultra-wide desktop monitors.
- **Expertise Visibility**: Technical features and stacks are intelligently revealed based on device capabilities (hover for desktop, default for mobile).

### 🛠 Technical Core
- **Framework**: React 18 with Vite for ultra-fast development and optimized production builds.
- **Aesthetics**: Vanilla CSS + Tailwind for a bespoke glass-morphic design system.
- **Componentry**: Custom logic-driven components including the "Vault" game archive and "Technical Spines" for process visualization.

## 🛠 Tech Stack

| Layer | Technologies |

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sivchheng16/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Initialize the development server:
   ```bash
   npm run dev
   ```

## 📐 Architecture Note
The project follows a **Modular Component Architecture**. Key sections are divided into highly reusable segments:
- `src/components/`: Shared UI primitives and high-level layout blocks.
- `src/pages/`: Content-dense views managed via `react-router-dom`.
- `src/constants.ts`: Centralized data store for projects, services, and topics.
